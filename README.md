> 专为《龙之谷怀旧服》设计的智能石板洗练辅助工具

## ✨ 项目简介

石板洗练助手是一款通过图像识别技术，自动检测游戏内石板词条颜色并执行洗练操作的辅助工具。它可以帮助您高效地达成目标词条组合，释放双手，专注于游戏策略与乐趣。

**核心设计原则**：采用**非侵入式**的屏幕图像分析方式，不修改游戏内存、不注入进程，以更安全的方式提供自动化辅助。

## 📌 主要功能

- **智能区域检测**：支持设置最多6个独立检测区域，全面覆盖石板词条位置。
- **双模式洗练策略**：

  - **基础模式**：设定目标红色词条数量，达标即停。
  - **高级模式**：为每个区域单独指定颜色需求（红/无），实现精准词条规划。
- **稳定可靠**：内置动画等待机制和异常处理，避免因网络延迟或卡顿导致的误操作。
- **便捷交互**：支持热键（F2）快速暂停/继续，实时日志与状态反馈。
- **性能优化**：包含图像缓存和内存管理，长期运行稳定。

## 🚀 快速开始

### 环境要求

- **操作系统**：Windows 10 或更高版本
- **Python 版本**：Python 3.7 或更高版本

### 安装步骤

1. **克隆或下载本项目**到本地。
2. **安装必备的Python库**，在项目根目录打开命令行，执行以下命令：
    bash

```
pip install -r requirements.txt
```

如果未提供 `requirements.txt` 文件，请手动安装核心依赖：

bash

```
pip install pyautogui pillow numpy pynput
```

**运行程序**：双击运行 `stone_washing_assistant.py`，或在命令行中执行：

bash

```
python stone_washing_assistant.py
```

## 📖 使用指南

1. **启动与配置**：

    - 运行程序，主界面将打开。
    - 点击  **“选择洗练按钮位置”** ，按提示将鼠标移动到游戏内的洗练按钮上，按下**空格键**确认。
    - 依次点击  **“区域1”** 、 **“区域2”** ... 等按钮，在屏幕上框选出每个词条的显示区域。
2. **设置洗练目标**：

    - 在  **“最低红色词条数量”**  下拉框中设置您的目标。
    - 如需更精细控制，可勾选  **“启用高级洗练目标策略”** ，并为每个区域指定需要洗出的颜色（“红”或“无”）。
3. **开始与监控**：

    - 点击  **“开始洗练”** ，程序将自动点击洗练按钮并分析结果。
    - 所有操作和结果会实时显示在右侧日志中。
    - 按下 **F2** 键可以随时暂停或继续洗练过程。
4. **达成目标**：

    - 当洗练结果满足您设定的所有条件时，程序会自动**弹窗提示**、**播放提示音**并停止。

## ⚙️ 技术实现简述

- **核心逻辑**：通过 `pyautogui` 控制鼠标点击，使用 `PIL`（Pillow）库进行屏幕截图。
- **颜色识别**：对指定区域的截图进行像素级分析，通过RGB颜色范围和容差判断是否为红色词条。
- **状态同步**：采用多线程设计，确保UI响应与洗练循环互不阻塞。
- **配置持久化**：用户设置（坐标、策略）会自动保存为 `config.json` 文件。
